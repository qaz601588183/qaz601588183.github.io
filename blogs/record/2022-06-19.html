# js赋值问题

  • 这里会先执行 a = { n:1 }
  • 再执行 b = a
  • 然后a.x = a = { n: 2 }
  • 先执行a.x会在之前a的存储处创建一个x的属性
  • 因为a.x需要右侧的值，所以会先执行 a = { n:2 } 让a指向一个新的{ n :2 }
  • 然后在之前的a.x指向新的{ n :2 } 因为b变量指针指向之前a的地址处=> b.x = 2
  • 而新的a没有x属性，所以会给出undefined
  • 画图理解更快！
var a = { n: 1 }
var b = a
a.x = a = { n: 2 }
console.log(a.x) // undefined
console.log(b.x) // { n:2 }
